🎬 CineFlowDiscover cinema. Anywhere. Anytime. > An immersive, beautifully designed movie discovery web application featuring interactive 3D WebGL backgrounds, dynamic bento grid layouts, and real-time cinematic data.✨ Features🌊 Liquid Ether WebGL Background: A highly optimized, interactive fluid simulation running in the background powered by Three.js. It dynamically reacts to your mouse movements.🍱 Dynamic Bento Grid Layout: Movie posters aren't just in a boring list. They are arranged in a responsive, magazine-style bento grid that intelligently spans columns and rows.🔍 Global Search & Discovery: Instantly search for any movie or browse through categorized genres (Action, Sci-Fi, Drama, Comedy, Horror) and Web Series.🍿 "Where to Watch" Intel: Never wonder where a movie is streaming. Get instant, region-specific availability data (Netflix, Prime, Hulu, etc.).🌓 Seamless Theme Switching: A beautifully crafted custom color palette (cineMint, cineGreen, cineForest, cineDark) that fully supports Dark and Light modes.🔠 Variable Proximity Typography: Interactive landing page typography that reacts and shifts its weight and optical size based on your cursor's proximity.⚡ Zero Dependencies Build: Built primarily with Vanilla JavaScript and HTML, utilizing CDNs for styling and 3D rendering to ensure lightning-fast load times.🛠️ Tech StackFrontend: HTML5, Vanilla JavaScript, Custom CSSStyling: Tailwind CSS (via CDN)3D Rendering: Three.js (WebGL)Data API: TMDB (The Movie Database) APITypography: Google Fonts (Inter, Bebas Neue, Roboto Flex)Icons: FontAwesome 6🚀 Getting StartedSince CineFlow is built as a highly optimized, single-file application without the need for complex build steps or bundlers, getting it running locally is incredibly simple.PrerequisitesA modern web browser (Chrome, Firefox, Safari, Edge).(Optional but recommended) A basic local web server like Live Server for VS Code to prevent any strict browser CORS issues when fetching data.InstallationClone the repository:git clone [https://github.com/Rachit-Mohanty/cineflow.git](https://github.com/Rachit-Mohanty/cineflow.git)
cd cineflow
Run the App:Simply double-click the index.html file to open it in your browser, or serve it via a local development server (e.g., using VS Code Live Server).🔑 API ConfigurationThis application uses The Movie Database (TMDB) API to fetch movies, genres, and streaming provider data
